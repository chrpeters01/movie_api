myFlix API

This web application provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Buisness Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding a list of all movies
Return a list of ALL users to the user /users GET None A JSON object holding a list of all users
Return data about a single movie by title to the user /movies/:Title GET None A JSON object holding data about a single movie title, including Genre/Director details. Example: { "Genre": { "Name": "Comedy", "Description": "Comedy is a film genre that..." }, "Director": { "Name": "Adam McKay", "Bio": "Adam McKay is...", "Birth": "1968", "Death": "" } }
Return data about a genre (description) by name/title: /movies/genres/:genreName GET None A JSON object holding data about the genre. Example: "Genre": { "Name": "Action", "Description": "Genre is..." }
Return data about a director (bio, birth year, death year) by name: /movies/directors/:directorName GET None A JSON object holding data containg the director's biography, birth year, and death year (if applicable) Example: { "Name": "J.J. Abrams", "Bio": "J.J. Abrams is ...", "Birth": "1965", "Death": "1966" }
Allow new users to register /users POST None A JSON object holding new user to add. Structure: { "Name": "Katie", "Password": "Password1", "Email": "Katie07@gmail.com", "Birthdate": "05/02/1980" } A JSON object holding data about a user that was added, inluding an ID: { "Name": "Katie", "Password": "Password1", "Email": "Katie07@gmail.com", "Birthdate": "1980-05-02T00:00:00Z", "FavoriteMovies": [], "_id": "656f7748f1edb95eb42dfd00", "__v": 0 }
Allow users to update their user info (username, password, email, date of birth) /users/:userId PUT None A JSON object holding user info that should be changed. Structure: { "Password": "Password2", "Email": "Thomas23@gmail.com" } A JSON object holding data about a user that was updated: { "_id": "656f6b24f1edb95eb42dfcfd", "Username": "Tom ", "Password": "Password2", "Email": "Thomas23@gmail.com", "Birthdate": "1975-04-25T00:00:00Z", "favorite_movies": [ "656f6b24f1edb95eb42dfcfd", "656f5482f1edb95eb42dfcf8" ], "__v": 0 }
Allow users to add a movie to their list of favorites /users/:userId/favorites POST None A JSON object holding data about user's favorite list. Structure: { "_id": "656f6b24f1edb95eb42dfcfd", "Username": "Tom ", "Password": "Password2", "Email": "Thomas23@gmail.com", "Birthdate": "1975-04-25T00:00:00Z", "favorite_movies": [ "656f6b24f1edb95eb42dfcfd", "656f5482f1edb95eb42dfcf8" ], "__v": 0 }
Allow users to remove a movie from their list of favorites: /users/:userId/favorites/:movieId DELETE None A JSON object holding data about user's favorite list. Structure: { "_id": "656f6b24f1edb95eb42dfcfd", "Username": "Tom ", "Password": "Password2", "Email": "Thomas23@gmail.com", "Birthdate": "1975-04-25T00:00:00Z", "favorite_movies": [], "__v": 0 }
Allow existing users to deregister: /users/:userId DELETE None A text message indicating whether a user was successfully removed.

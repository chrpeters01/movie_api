myFlix API

This web application provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Buisness Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding a list of all movies
Return data about a single movie by title to the user /movies/:title GET None A JSON object holding data about a single movie
Return data about a genre (description) by name/title: /genres/:name GET None A JSON object holding data about the genre
Return data about a director (bio, birth year, death year) by name: /directors/:name GET None A JSON object holding data containg the director's biography, birth year, and death year (if applicable)
Return a list of ALL users /users GET None A JSON object holding data about all users
Allow new users to register /users POST None A JSON object holding new user data
Allow users to update their user info (username): /users/:userId PUT None A JSON object holding user info that should be changed
Allow users to add a movie to their list of favorites /users/:userId/favorites POST None A JSON object holding updated user info
Allow users to remove a movie from their list of favorites: /users/:userId/favorites/:movieId DELETE None A JSON object holding updated user info
Allow existing users to deregister: /users/:userId DELETE None A text message indicating whether a user was successfully removed.
